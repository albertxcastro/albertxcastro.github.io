The null hypothesis (H0) is that no difference exists between means at a p-value of 0.05. However, looking at the two-tailed p-value of 0.007545995, we can see that it is lower than the alpha p-value (0.05), so we can reject the null hypothesis. Still, the difference is statistically significant and is not due to random chance, given the low p-value obtained in the test.
Excercise 8.5
In excersice 8.4, a two-tailed test was undertaken of wether the population mean impurity differs between the two filtration agents in data set G
Question: Supose instead a one-tailed test had been conducted to determine wether Filter Agent 1 was more effective. What would your conclusions have been?
My answer
Maybe my conclusions will be now biased because I already observed the results of the two-tailed test. However, if a one-tailed test had been conducted, we would have set a directional null hypothesis, for example, that Agent 1 is more effective than Agent 2 (Agent 1 has a lower mean impurity value), and the hipotheses would have been: H0: 𝝁Agent1 >= 𝝁Agent2 and H1: 𝝁Agent1 < 𝝁Agent2. In this case, the result is consistent with H1, and given that the one-tail p-value is lower than alpha at 0.05, we conclude that the result is significant, rejecting H0.
