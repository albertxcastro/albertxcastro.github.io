First order logic

First-order logic is a fundamental framework in symbolic logic, enabling the precise and rigorous representation of statements concerning objects and their relationships. Its extensive application spans diverse fields, including mathematics, computer science, philosophy, and linguistics. This logic system articulates statements effectively using quantifiers, variables, predicates, and logical connectives. (Enderton, 2001).

There are different components utilized for representing statements:

  1. Variables: Variables are placeholders that can take different values in specific domains. They are typically represented with lowercase letters such as x and y.
  2. Quantifiers: Quantifiers express the scope of a variable. There are two main quantifiers used in first-order logic:
    • Universal (∀): Represents "for all" or "for every". It expresses something true for all elements in a domain.
    • Existential (∃): Represents "there exists". It expresses that at least one element exists in a value domain holding a true value.
  3. Predicates: Predicates are propositions containing variables that transform into meaningful statements when variables are assigned values. They are typically represented with uppercase letters and can take one or more arguments, for instance, P(x), D(x, y).
  4. Logical connectors: Logical connectors combine predicates and build more complex statements.
    • AND (∧): Represents logical conjunction.
    • OR (∨): Represents logical disjunction.
    • NOT (¬): Represents negation.
    • Implication (→): Represents "if...then" statements.
    • Biconditional (↔): Represents "if and only if" statements.

Examples of first-order logic statements

Let P(x) be the predicate "x can fly", and Q(x) "x is carnivore"
      Universal statement: ∀x: P(x) → "Every x can fly."
      Existential statement: ∃x: Q(x) → "There exists x that is a carnivore."
